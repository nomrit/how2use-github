# how2use-github
忘れそうなので、githubの使い方をまとめる

## initial

最初にgithubでinitial repositoryを作成する。

## move directory

リポジトリ作るdirectoryに移動。例えば、
```
$ mkdir work
$ cd work 
```
## clone

git clone.
例えばこのrepositoryであれば、

```
$ git clone https://github.com/nomrit/how2use-github
```

## ローカルリポジトリをgithubにプッシュ

```
$ git push origin master
```
